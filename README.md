
# Tutoriat Programarea Calculatoarelor I
## Informatii 

    E - Nota examen (10 pct maxim)
    L - Nota laborator (10 pct maxim)
    S - Nota seminar (1 pct maxim)
    P - Numarul Prezente seminar
    R - Numarul de Raspunsuri seminar
    S = P*0.1 + R*0.1;
    N = min(10, E*0.6 + L*0.4 + S);
